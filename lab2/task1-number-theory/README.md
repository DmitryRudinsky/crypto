# Task 1: Number Theory Service

## Описание

Stateless-сервис для базовых операций теории чисел. Все алгоритмы реализованы с нуля без использования встроенных методов Java (кроме BigInteger для работы с большими числами).

## Реализованный функционал

### 1. Символ Лежандра
`legendreSymbol(BigInteger a, BigInteger p)` → `int`

Вычисление символа Лежандра (a/p) для простого p через формулу Эйлера: a^((p-1)/2) mod p.

Возвращает:
- 1 если a - квадратичный вычет по модулю p
- -1 если a - квадратичный невычет по модулю p  
- 0 если a ≡ 0 (mod p)

### 2. Символ Якоби
`jacobiSymbol(BigInteger a, BigInteger n)` → `int`

Вычисление символа Якоби (a/n) для нечетного n > 1. Реализован алгоритм с использованием квадратичного закона взаимности.

Возвращает: -1, 0 или 1

### 3. НОД (алгоритм Евклида)
`gcd(BigInteger a, BigInteger b)` → `BigInteger`

Классический алгоритм Евклида для нахождения наибольшего общего делителя двух чисел.

### 4. Расширенный алгоритм Евклида
`extendedGcd(BigInteger a, BigInteger b)` → `ExtendedGcdResult`

Находит НОД и решает уравнение Безу: ax + by = gcd(a, b)

Возвращает объект с полями:
- `gcd` - наибольший общий делитель
- `x` - коэффициент для a
- `y` - коэффициент для b

### 5. Модульное возведение в степень
`modPow(BigInteger base, BigInteger exponent, BigInteger modulus)` → `BigInteger`

Быстрое возведение в степень по модулю методом binary exponentiation (возведение в квадрат).

Вычисляет: base^exponent mod modulus

## Тесты

`NumberTheoryServiceTest.java` - 37 тестов, покрывающих все методы:

- **Символ Лежандра** (9 тестов): проверка квадратичных вычетов и невычетов для различных простых модулей
- **Символ Якоби** (7 тестов): проверка на составных нечетных модулях, включая классические примеры
- **НОД** (7 тестов): проверка корректности для различных пар чисел, включая граничные случаи (0, взаимно простые числа, большие числа)
- **Расширенный НОД** (8 тестов): проверка правильности НОД и верификация уравнения Безу для всех случаев
- **Модульное возведение** (6 тестов): проверка на малых и больших степенях, включая степень 0 и большие модули

Все тесты проходят успешно.

## Демонстрация

`Demo.java` - примеры использования каждого метода:

1. Символ Лежандра: L(2,7), L(3,11), L(5,11)
2. Символ Якоби: J(1001,9907), J(19,45), J(8,21)
3. НОД: gcd(48,18), gcd(1071,462), gcd(123456789,987654321)
4. Расширенный НОД: extgcd(240,46), extgcd(1071,462) с верификацией уравнения Безу
5. Модульное возведение: 3^7 mod 13, 2^100 mod 1000000007, 123456789^987654321 mod 1000000009

## Запуск

```bash
./run-demo.sh   # сборка + запуск демонстрации
./run-tests.sh  # сборка + запуск тестов
./build.sh      # только сборка
```
